# Tharaniesh Portfolio

Production-ready single-page portfolio built with React, Vite, TypeScript, TailwindCSS, Framer Motion, and React Three Fiber.

## Stack
- React 18 + TypeScript
- Vite
- TailwindCSS
- Framer Motion
- Three.js via React Three Fiber + Drei

## Setup
1. Install dependencies:
   npm install
2. Start local development server:
   npm run dev
3. Create production build:
   npm run build
4. Preview production build:
   npm run preview
